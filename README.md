# Snake-cpp #

## Classic snake game in a simple windows terminal version ##<hr>
## Instruction ##
### Just clone the repository and build the main.cpp file ###
### Example : ###
**git clone https://github.com/Andevson/snake-cpp.git**<br>
**cd snake-cpp**<br>
**g++ ./main.cpp -o main.exe**<br>
